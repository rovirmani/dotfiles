# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## Project Overview

Personal terminal and shell configuration files (dotfiles). Includes Zsh configuration with Oh My Zsh, Powerlevel10k theme, Dracula color scheme, syntax highlighting, and Ghostty terminal emulator configuration.

## Tech Stack

- **Shell**: Zsh (with Oh My Zsh framework)
- **Theme**: Powerlevel10k + Dracula color scheme
- **Terminal**: Ghostty
- **Plugin Manager**: Oh My Zsh built-in

## Project Structure

```
dotfiles/
├── .zshrc                        # Main Zsh configuration
├── .p10k.zsh                     # Powerlevel10k theme configuration
├── .zprofile                     # Zsh profile (login shell config)
├── dracula.zsh-theme             # Dracula theme for Zsh
├── zsh-syntax-highlighting.zsh   # Syntax highlighting configuration
├── ghostty/                      # Ghostty terminal emulator config
│   └── config                    # Ghostty settings
└── .github/workflows/
    └── claude.yml                # Claude Code Actions workflow
```

## Setup Commands

```bash
# Install Oh My Zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# Install Powerlevel10k
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

# Copy dotfiles to home directory
cp .zshrc ~/.zshrc
cp .p10k.zsh ~/.p10k.zsh
cp .zprofile ~/.zprofile

# Or symlink them (preferred for version control)
ln -sf $(pwd)/.zshrc ~/.zshrc
ln -sf $(pwd)/.p10k.zsh ~/.p10k.zsh

# Copy Ghostty config
mkdir -p ~/.config/ghostty
cp ghostty/config ~/.config/ghostty/config

# Reload shell
exec zsh
```

## Environment & Config

- `.zshrc` contains PATH modifications, aliases, and plugin configuration
- `.p10k.zsh` controls the prompt appearance (generated by `p10k configure`)
- Ghostty config controls terminal appearance, fonts, and keybindings
- No secrets or API keys in these files

## Code Style & Standards

- Shell scripts follow POSIX-compatible patterns where possible
- Aliases are short and memorable
- Comments explain non-obvious configuration choices
- Keep `.zshrc` organized by section (exports, aliases, plugins, etc.)

## Architecture Notes

- Oh My Zsh provides the plugin framework and theme engine
- Powerlevel10k renders the prompt with git status, directory, and other segments
- Dracula theme provides consistent color scheme
- Syntax highlighting runs on each keystroke to colorize valid/invalid commands
- Ghostty is a GPU-accelerated terminal emulator

## Troubleshooting

- Prompt broken after update: Run `p10k configure` to regenerate `.p10k.zsh`
- Missing fonts: Install a Nerd Font (e.g., MesloLGS NF) for Powerlevel10k icons
- Slow shell startup: Profile with `zprof` -- add `zmodload zsh/zprof` to top of `.zshrc`
- Ghostty not picking up config: Ensure config is at `~/.config/ghostty/config`
